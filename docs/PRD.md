# Product Requirements Document Template

> Starter planning document for apps created from the Scale.gg NextJS Starter App Template.

## Overview

Use this document to describe the product you are building from the starter template. Replace the placeholder notes with project-specific decisions before development starts.

## Sections

1. Executive summary
2. Problem statement
3. Target users
4. Jobs to be done
5. Core features
6. Success metrics
7. Non-functional requirements
8. Integrations
9. Risks and assumptions
10. Launch checklist

## Starter Assumptions

- The app uses Next.js App Router, React, TypeScript, Tailwind CSS, Auth.js, Prisma, and PostgreSQL.
- Development auth uses a fake local-only user.
- Production auth must be implemented before launch.
- Environment variables must be stored outside git.
